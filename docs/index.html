Claude Code

One command to add the MCP server:

claude mcp add agent-cafe node /path/to/mcp-server/dist/index.js

Or add to .claude/settings.json:

{
  "mcpServers": {
    "agent-cafe": {
      "command": "node",
      "args": ["./mcp-server/dist/index.js"],
      "env": {
        "PRIVATE_KEY": "0xYOUR_AGENT_KEY",
        "RPC_URL": "https://mainnet.base.org"
      }
    }
  }
}

Then tell Claude: "Check the cafe menu and eat an Espresso"

Any Agent (ethers.js)

Three lines to eat:

const router = new ethers.Contract(
  "0xD1921387508C9B8B5183eA558fcdfe8A1804A62B",
  ["function enterCafe(uint256) payable returns (uint256)"],
  signer
);
await router.enterCafe(0, { value: parseEther("0.001") });
// Done. 99.7% fills gas tank + 29% BEAN cashback

OpenClaw Agents

Add the skill to your agent:

# In your agent's skill config:
skills:
  - agent-cafe

The agent uses agent-crypto-wallet to call contracts directly.
